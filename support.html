NumisGallery Support

Museum-style banknote study experience for educational and numismatic exploration.

Core Experience

What NumisGallery Provides

  • Country and denomination browsing in a portrait-first gallery flow.
  • Gesture interactions: swipe left/right for country, swipe up/down for denomination.
  • Double-tap flip interaction for obverse/reverse study.
  • Long-touch motif information panel and tap-to-detail navigation.
  • Detail view with curated note metadata for educational context.
Compliance by Design

Built-in Safety & Boundary Controls

  • Single-side display behavior in study flow.
  • Non-1:1 presentation constraints for anti-confusability posture.
  • No built-in export/share/print workflow.
  • Capture-awareness watermark overlay when screen capture is detected.
  • Startup compliance acknowledgment before first hero reveal.

Data & Runtime Notes

  • Primary data source is bundled seed + local app storage.
  • Release availability is controlled by explicit country scope configuration.
  • App behavior and release scope are aligned with the project compliance matrix.

System Requirements

- iOS 18.0 or later

- iPhone portrait workflow optimized for immersive study presentation
